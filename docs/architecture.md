# Texas Hold'em AI 系统架构设计
=====================================

## 概述
----

本架构设计的核心思想是将**策略逻辑**与**游戏平台**解耦，使得同一套策略可以在不同的平台（浏览器自动化、本地竞技场、Web 对战等）上运行。

## 核心模块说明
------------

### 1. 核心层 (Core)

核心层定义了整个系统的通用接口和数据结构。

#### src/core/interfaces.py
- **GamePlatform**: 游戏平台抽象接口
  - `initialize()`: 初始化平台
  - `get_game_state()`: 获取游戏状态
  - `execute_action()`: 执行动作
  - `wait_for_my_turn()`: 等待轮到自己
  - `wait_for_hand_start()`: 等待新手牌开始
  - `shutdown()`: 关闭平台
  
- **PlayerAgent**: 玩家代理接口
  - `decide_action()`: 根据游戏状态决策
  - `handle_event()`: 处理游戏事件
  - `reset()`: 重置状态
  
- **GameRunner**: 游戏协调器
  - 连接 PlayerAgent 和 GamePlatform
  - 协调整个游戏流程

- **GameState**: 通用游戏状态
- **GameAction**: 游戏动作
- **ActionType**: 动作类型枚举
- **Player**: 玩家信息

#### src/core/events.py
- **EventBus**: 事件总线
- **EventType**: 事件类型枚举
- **GameEvent**: 游戏事件

#### src/core/adapters.py
- **BrainToAgentAdapter**: 将现有的 Brain 策略到 PlayerAgent 的适配器
- **create_strategy_agent()**: 策略工厂函数

### 2. 策略层 (Brain)

策略层包含各种德州扑克策略实现。

### 3. 平台层 (Platforms)

平台层提供不同游戏环境的实现。

#### BrowserPlatform (`src/platforms/browser/`)
- 连接到 ReplayPoker 网站
- 使用 Playwright 进行浏览器自动化
- 解析游戏状态并执行动作

#### ArenaPlatform (`src/platforms/arena/`)
- 本地模拟竞技场
- 用于策略测试和对战
- 支持多策略间快速模拟

#### WebPlatform (待实现)
- Web 对战平台
- 支持人机对战
- WebSocket 实时交互

## 使用示例
--------

### 浏览器模式

```python
import asyncio
from src.core import GameRunner, create_strategy_agent
from src.platforms import BrowserPlatform

async def main():
    # 创建策略代理
    agent = create_strategy_agent("range")
    
    # 创建浏览器平台
    platform = BrowserPlatform(headless=False)
    
    # 创建游戏协调器
    runner = GameRunner(platform, agent)
    
    # 运行游戏
    await runner.run(max_hands=100)

asyncio.run(main())
```

### 竞技场模式

```python
import asyncio
from src.core import GameRunner, create_strategy_agent
from src.platforms import ArenaPlatform

async def main():
    # 创建策略代理
    agent = create_strategy_agent("balanced")
    
    # 创建竞技场平台
    platform = ArenaPlatform()
    
    # 创建游戏协调器
    runner = GameRunner(platform, agent)
    
    # 运行游戏
    await runner.run(max_hands=1000)

asyncio.run(main())
```

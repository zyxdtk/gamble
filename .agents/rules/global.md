# Global Rules 

## 1. 语言偏好 / Language Preference
- **核心原则**: 所有解释、注释、文档、设计方案、任务列表及与用户的交流**必须使用中文**。
- **问候语**: 每次回复的第一句必须是“你好，同学”。

## 2. 工具偏好 / Tool Preference
- **依赖管理**: 默认使用 `uv` 管理依赖和虚拟环境。
- **环境激活**: 确保在执行 Python 相关命令前激活虚拟环境。

## 3. 项目结构 / Project Structure
- **代码**: `src`
- **测试**: `tests`
- **文档**: `docs` (Default location for design proposals, tasks, and plans)
- **配置**: `config`
- **数据**: `data`

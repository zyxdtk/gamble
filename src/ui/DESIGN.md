# UI 模块设计文档

该目录目前主要专注于浏览器覆盖层（HUD）等可视化的扩展。

## 1. 浏览器 HUD (Heads-Up Display)

### 目标
在 ReplayPoker 游戏页面上直接叠加显示 AI 分析结果、胜率和决策建议。

### HUD 设计

| 属性 | 规格 |
|------|------|
| 位置 | 页面右上角，fixed 定位 |
| 样式 | 半透明黑色背景 + 绿色/黄色高亮文字，monospace 字体 |
| 交互 | 可拖拽移动、可关闭（X 按钮） |
| 更新 | 每个游戏 Tick 自动刷新 |

### 显示内容
- 标题：「AI 顾问 ✥」
- 手牌分析：当前牌型和强度（如「对 K」）
- 决策建议：**加注/跟注**（黄色加粗高亮）
- 胜率：青色加粗
- 对手分析：各对手范围评估

### 预期效果
```
┌─────────────────────────────┐
│ AI 顾问 ✥              × │
├─────────────────────────────┤
│ 翻牌后: 对 K - 跟注/加注    │
│ 胜率 (对 3 位对手): 72.5%  │
│ 【对手分析】                │
│   - 座位 2 (call): 宽松/投机│
│   - 座位 4 (raise): 紧凶    │
└─────────────────────────────┘
```

### 技术实现
- 使用 `page.evaluate()` 注入 JavaScript。
- `z-index: 2147483647` 确保在最顶层。
- 内容更新：转义反引号和 `${` 字符，使用模板字符串安全注入。

### 已解决的问题
1. **重复事件监听器导致 SyntaxError**：将 HUD 逻辑提取为独立模块，添加防御性检查 `if (document.getElementById('ai-hud')) return;`
2. **特殊字符导致注入失败**：统一转义反引号和 `${` 字符。
